# 🤖 AI Resume Intelligence System

An AI-powered web application built using Flask that helps users create, analyze, and improve their resumes with smart job matching and career guidance.

---

## 🚀 Features

- 📄 Resume Builder (Generate PDF)
- 📊 Resume Analyzer (Score & Feedback)
- 🎯 Job Match System (Match resume with job description)
- 🧠 Career Suggestions (Skills & project guidance)
- 📥 Downloadable PDF Reports
- 💬 AI Assistant (Free chatbot for resume help)

---

## 🛠 Tech Stack

- Python
- Flask
- HTML, CSS, JavaScript
- ReportLab (PDF generation)
